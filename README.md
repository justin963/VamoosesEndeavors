# Vamoose's Endeavors

Track Housing Endeavor progress across all your characters in World of Warcraft: Midnight.

## Version 1.2.0 - Theme Update

### What's New in 1.2.0

- **5 Theme Options** - Solarized Dark, Solarized Light, Gruvbox Dark, Gruvbox Light, and Everforest
- **Quick Theme Toggle** - "T" button in title bar cycles through all themes
- **Live Theme Switching** - All UI elements update instantly
- **Cleaner Title Bar** - Borderless design with centered title

### Features

- **Endeavor Task List** - View all housing endeavor tasks with completion status
- **Leaderboard Tab** - See contribution rankings for all neighborhood members
- **Activity Tab** - Track recent neighborhood activity and top tasks
- **Community Coupons** - Displays coupon count and per-task rewards
- **Repeatable Indicators** - Green refresh icon shows which tasks can be repeated
- **Progress Tracking** - See X/Y progress for multi-step tasks
- **Point Values** - Gold badge showing endeavor points per task
- **Season Progress** - Progress bar with milestone markers and days remaining
- **5 Themes** - Solarized Dark/Light, Gruvbox Dark/Light, Everforest
- **Minimap Button** - Quick access to the window

### Installation

1. Download and extract to `World of Warcraft/_retail_/Interface/AddOns/`
2. Folder should be named `VamoosesEndeavors`
3. Type `/reload` or restart WoW
4. Type `/ve` to open

### Commands

- `/ve` - Toggle main window
- `/ve debug` - Toggle debug mode
- `/ve refresh` - Force refresh endeavor data
- `/ve theme` - Toggle theme

### Requirements

- World of Warcraft: Midnight (12.0.0+)
- Access to Housing (required level)
